XOR Problem with Neural Network

This sketch solves the XOR problem with a Neural Networkas demonstrated on The Coding Train in Coding Challenge 92: XOR Problem by Dan Shiffman on YouTube.

This sketch uses p5.js and the Toy Neural Network library from the video. The left side is the solution space for the problem in 2D and the right side is the graph of the error in the output during training. Note that this graph is only updated once per frame.

Press space to stop and start the learning. Use sliders to adjust the neural network.
